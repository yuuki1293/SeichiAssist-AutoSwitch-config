# SeichiAssist-AutoSwitch-config
ギガンティック☆整地鯖でAutoSwitchを使う際の不具合の一部をなくすconfigファイル(非公式)

## 修正される不具合
・砂や砂利を木の棒やガチャりんごで掘る  
・木の葉や蔦をPLUTOで掘る

## 導入方法
1. Codeからzipファイルをダウンロードし、展開する。
2. 起動構成のフォルダーを開く。
3. `config`フォルダーの中の`AutoSwitch_Overrides.txt`をダウンロードしたものに置き換える。
